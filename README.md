# SafeW / SafeX Brand and Domains

本仓库记录 SafeW / SafeX 相关名称、包名、公开分发线索、搜索矩阵截图与洗白话术。

它的用途不是重复证明恶意代码本身，而是解释为什么普通中文用户搜索“SafeW 安全吗”时容易被推广站、矩阵站和软文误导。
